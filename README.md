# 淘宝商品高清图保存助手

一个 Windows 本机桌面工具，用可见的真实浏览器窗口辅助浏览淘宝搜索结果，并保存页面真实暴露出的商品主图和详情图。

当前版本偏向服装商品：每个商品最多保存 5 张图片，优先保留正面、侧面、背面等服装主体图，并尽量过滤平台证照、Logo、参数表、促销图、评论图和组图。

## 合规边界

- 工具用于个人资料整理和手动浏览辅助。
- 遇到登录、验证码、安全校验、滑块验证时，需要用户在浏览器里手动处理。
- 不包含验证码绕过、指纹伪装、代理池、隐藏自动化痕迹或高频并发采集能力。
- 请控制使用频率，并遵守淘宝网站条款、版权要求和当地法律法规。

## 当前设计

桌面工具会启动一个你选择的真实浏览器程序，例如 Chrome 或 Edge，并使用项目目录里的 `browser-profile/` 保存登录状态。第一次使用时，你需要在这个被工具打开的浏览器窗口里登录淘宝；之后通常会复用这个登录状态。

这种方式不依赖 Chrome 扩展，也不接管你已经打开的普通 Chrome 窗口，因此更稳定，也不会要求关闭所有 Chrome。

## 安装

需要 Python 3.10 或更新版本。

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
pip install -e .
```

如果系统里没有 `python` 命令，请先安装 Python，并在安装时勾选 `Add python.exe to PATH`。

## 运行

```powershell
python -m taobao_image_saver
```

也可以运行：

```powershell
.\run_taobao_image_saver.bat
```

## 使用流程

1. 确认“浏览器程序”指向 `chrome.exe` 或 `msedge.exe`。
2. 先点击 `登录准备`，在打开的浏览器窗口里登录淘宝。
3. 登录成功后，关闭这个登录准备窗口。
4. 在工具里输入关键词。
5. 设置最大商品数、操作间隔、每页滚动次数和保存目录。
6. 点击 `开始自动浏览保存`。
7. 如果浏览器出现登录或安全验证，请手动完成；工具会等待并继续。
8. 图片会保存到输出目录，每个商品一个文件夹。

如果主图角度不足，工具会继续到详情页寻找更像服装主体展示的图片；若仍不足 3 张，会在 `metadata.json` 里记录原因。

输出示例：

```text
output/
  商品标题/
    metadata.json
    images/
      main_001.jpg
      detail_001.jpg
```

`metadata.json` 会记录标题、链接、价格、店铺名、图片文件列表、采集时间和失败原因。

## 按钮说明

- `开始自动浏览保存`：自动打开淘宝搜索页、滚动、打开商品页并保存图片。
- `登录准备`：用同一个工具资料目录普通打开淘宝，用于提前登录。登录后请关闭该窗口再开始自动任务。
- `仅打开搜索页`：只用所选浏览器打开搜索页，不会自动保存图片。
- `暂停` / `继续`：临时暂停或恢复任务。
- `停止`：结束当前任务。
